About Me

שמי מאיר לוי, בן 78, מתגורר בחיפה עם רעייתי מרים. אני אב לשלושה ילדים וסב לשבעה נכדים מקסימים. נולדתי בקיבוץ בגליל התחתון בשנת 1947, וגדלתי בין שדות, מטעים וספרים. אהבתי הגדולה להיסטוריה התחילה כבר בילדותי, והובילה אותי ללמוד תואר ראשון ושני באוניברסיטת חיפה. במשך 35 שנה שימשתי כמורה להיסטוריה ואזרחות בבית ספר תיכון, שם העברתי אלפי שיעורים, הדרכתי משלחות, וחינכתי דורות של תלמידים לאהבת הארץ והעבר שלה. לאחר פרישתי, אני ממשיך בהתנדבות בבתי אבות, מעביר הרצאות לנוער, ומקדיש זמן לתחביבים האהובים עליי. התחביבים שלי כוללים קריאה (בעיקר היסטוריה), גינון, טיולים רגליים, ושמיעת שירים ישנים של אריק איינשטיין. אני גם אוהב להכין ריבות תוצרת בית ולחלק למשפחה. הדמות שהשפיעה עליי יותר מכל היא דוד בן גוריון – אדם של חזון, צניעות ומעשים.

המוטו שלי הוא: "עם שאינו יודע את עברו, הווה שלו דל ועתידו לוט בערפל"

Q&A

1. הבדלים בין Web1 / Web2 / Web3

  • סוג האתרים והתוכן:
    • Web 1.0 – אתרים סטטיים עם תוכן קבוע שלא משתנה לעיתים קרובות.
    • Web 2.0 – אתרים דינמיים שמבוססים על תוכן מהמשתמשים (בלוגים, רשתות חברתיות).
    • Web 3.0 – אתרים חכמים עם בינה מלאכותית והתאמה אישית למשתמש.
  • תפקיד המשתמש:
    • Web 1.0 – המשתמש הוא רק קורא מידע (צרכן פסיבי).
    • Web 2.0 – המשתמש משתתף פעיל – יוצר תוכן, מגיב, משתף.
    • Web 3.0 – המשתמש מקבל תוכן מותאם אישית ויכול לתקשר עם מערכות חכמות.
  • שיטות חיפוש וטכנולוגיה:
    • Web 1.0 – חיפוש בסיסי ולא מדויק, ללא ניתוח משמעות.
    • Web 2.0 – חיפוש טוב יותר, עם תוצאות מגוונות כמו סרטונים, פוסטים ובלוגים.
    • Web 3.0 – חיפוש סמנטי חכם, שמבין את כוונת המשתמש ומאחד מידע ממקורות רבים.

2. תרחיש שימוש – חיפוש "תופעות לוואי של תרופות"

  • Web 1.0 – המשתמש נכנס לאתר רפואי סטטי שמציג טקסט כללי בלבד, ללא אינטראקציה או תגובות.
  • Web 2.0 – המשתמש מוצא מגוון תכנים כמו פוסטים, סרטונים, תגובות בפורומים או בלוגים.
  • Web 3.0 – המשתמש מקבל תשובה מותאמת אישית שמבוססת על ניתוח סמנטי של מידע רפואי ממקורות רבים, בעזרת בינה מלאכותית.
